# Number Memory Game

## Description
In introduction, 
this project is all about programming a number memory game
that can be played on an Mbed Pico Extension board. An LCD is attached to on the board to
display the numbers the user has to memorise. It also includes
a keypad that allows the user to type numbers between 0 - 9 and
letters from A - F. The majority of this program is in C++ and a tiny bit
is in C.

## Getting Started

### Dependencies

* The arduino IDE is recommended to program the game
as it will contain the packages that you will
need to install 

* Before installing the program you will need
to have Mbed Pico OS Package installed 
in the arduino IDE.

* The OS version needs to be a Windows 10 and above
